# 🔓 Unlock Alert — 代币解锁预警 iOS App

> 币圈散户的「代币解禁日历」— 再也不用被团队/VC解锁砸盘了

---

## 这是什么？

追踪**代币解锁事件**的 iOS App。关注任意代币，在解锁前自动推送通知提醒你。

**核心功能：**
- 📋 所有代币的解锁时间表（按时间排序）
- 🔔 解锁前 7天/3天/1天本地通知提醒
- 💰 显示解锁金额（USD）和占总流通百分比
- ⭐ 关注列表，只看你关心的币
- 🔍 搜索代币（名称/符号）
- 📊 倒计时 + 解锁时间线

## 🚀 如何运行 App

### 准备工作
1. 一台 Mac（安装 Xcode 15+）
2. 可选：Apple Developer 账号（$99/年，如需上架 App Store）

### 从 GitHub 下载

```bash
# 克隆项目
git clone https://github.com/zone1084/UnlockAlert.git
cd UnlockAlert/ios
```

### 在 Xcode 中运行

```
① 打开 Xcode → File → Open → 选择 ios/ 目录
② 等 Swift Package Manager 加载（几秒钟，这次不需要额外依赖）
③ 选择模拟器（iPhone 15 Pro 等）
④ Product → Run (Cmd+R)
```

> **第一次启动**会请求通知权限，点「允许」即可。

## 数据更新机制

- 数据文件：`data/tokens.json` 
- 更新方式：GitHub Actions **每天 UTC 2:00** 自动运行
- 更新内容：抓取 CoinGecko 最新价格，保持数据新鲜
- 手动触发：在 GitHub 仓库 → Actions → Daily Token Data Update → Run workflow
